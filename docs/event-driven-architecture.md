# 1. Purpose

This document defines the event-driven architecture of the Restaurant Intelligence Platform.

The goal is to create a system where services can communicate through events instead of direct dependencies.

The architecture improves:

- Scalability
- Reliability
- Maintainability
- Background processing
- AI workflow automation
